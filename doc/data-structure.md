#   社交网络信息融合-文档-数据结构篇

##  概述

+   由于信息融合需要收集大量原始数据，除了需要使用自定义数据结构之外，还需要大量使用关系数据库来存储 收集到的原始数据以用作分析。
+   处于方便和查询性能考虑， **数据采集阶段** 使用sqlite3作为关系数据库来使用，存储采集到的原始数据。
